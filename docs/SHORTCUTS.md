## Writing

| Shortcut | Action                          |
| -------- | ------------------------------- |
| `Ctrl+S` | Save current paper              |
| `Ctrl+N` | New paper                       |
| `Ctrl+B` | Bold selection                  |
| `Ctrl+I` | Italic selection                |
| `Ctrl+K` | Insert link                     |
| `Ctrl+F` | Find in editor (browser native) |
| `Tab`    | Insert 2 spaces                 |

## View Controls

| Shortcut             | Action                       |
| -------------------- | ---------------------------- |
| `Ctrl+Shift+T`       | Toggle theme (dark/light)    |
| `Ctrl++` or `Ctrl+=` | Increase font size           |
| `Ctrl+-`             | Decrease font size           |
| `F11`                | Toggle distraction-free mode |
| `Esc`                | Exit distraction-free mode   |

## Navigation

| Shortcut                 | Action                        |
| ------------------------ | ----------------------------- |
| `Ctrl+B` (not in editor) | Toggle sidebar                |
| `Ctrl+/`                 | Show keyboard shortcuts modal |
| `Esc`                    | Close modals/menus            |
